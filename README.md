# FinalProject-IoT

> ⚠️ **Warning:** This process may disable XAMPP (the software previously used to run MySQL). However, a similar setup can be achieved using MySQL Workbench.
>
To run and display this project on any laptop, follow the steps below:
>
> 1.- Download and configure **MySQL Server** and **MySQL Workbench** by following an online tutorial, such as the one below:  
> https://www.youtube.com/watch?v=_K2nOYwOq1E&ab_channel=ElRinc%C3%B3ndelHacker
>
> 2.- In your local **Visual Studio Code** terminal, install the required Python dependencies by running the following commands individually:  
> `pip install streamlit`  
> `pip install mysql-connector`  
> `pip install matplotlib`
>
> 3.- Run any of the `intentolocal.py` files locally using the following command:  
> `streamlit run "file_name_to_test.py"`
>
The following video may also be helpful as a visual guide:  
https://www.youtube.com/watch?v=zSRBVxvhriA&ab_channel=Beginner%27sCodeZone
